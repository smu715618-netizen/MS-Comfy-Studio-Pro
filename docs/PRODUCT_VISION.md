# 产品愿景 (Product Vision)

## 产品定位

MS Comfy Studio Pro 是一款**专业 AI 修图软件**。

不是整合包。不是启动器。不是工作流管理器。

AI 只是我们的能力层之一。用户感受到的应该是一个成熟、稳定的专业修图工具。

## 目标用户

| 用户群体 | 核心需求 |
|----------|----------|
| 商业摄影师 | 人像精修、批量处理、证件照生成 |
| 电商设计师 | 商品修图、白底图、促销海报 |
| 广告设计 | 创意设计、背景替换、文字排版 |
| 印刷行业 | 高精度修复、老照片恢复 |
| 工程行业 | 图纸增强、尺寸调整、标记添加 |
| 办公场景 | 简历制作、宣传册、批量文档输出 |
| 企业用户 | 标准化流程、批量生产、版本管理 |

## 目标平台

### 第一阶段（当前）
- Windows 11 + Intel Arc A750 8GB + 16GB RAM

### 预留扩展
- macOS + Apple Silicon Metal
- Linux + NVIDIA CUDA / AMD ROCm

## 技术架构原则

```
┌─────────────────────────────────────────────┐
│              GUI (普通用户界面)                │
├─────────────────────────────────────────────┤
│         AI Capability Layer (能力层)           │
├─────────────────────────────────────────────┤
│            AI Scheduler (调度器)               │
├─────────────────────────────────────────────┤
│        Engine Adapter (引擎适配层)              │
├─────────────────────────────────────────────┤
│       ComfyUI / Diffusers / ONNX / ...       │
└─────────────────────────────────────────────┘
```

关键原则：
- **禁止业务层直接调用 ComfyUI**
- 所有底层推理框架通过 Engine Adapter 统一接口调用
- 未来可无缝切换/添加推理引擎
- 程序与模型解耦
- 按需下载、按需安装、按需加载
